## Requirements


To run the Jupyter Notebooks on a Linux machine, make sure you have the following dependencies installed:


- Python 3.x
- Jupyter Notebook
- Required Python packages (NumPy, Pandas, Matplotlib, Scikit-learn, XGBoost, etc.)
